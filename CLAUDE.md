# West Him — сайт-каталог

Сайт казахстанского производителя бытовой и автохимии (г. Актау).
Цель: каталог продукции + информация для оптовых партнёров.

## Деплой

- **Живой сайт:** https://west-him.vercel.app
- **GitHub:** https://github.com/kereyevzh90-pixel/west-him
- Vercel автоматически деплоит при `git push` в `master`

```bash
git add .
git commit -m "описание"
git push
```

## Дизайн

- **Тема:** тёмная (фон `#000`), акцент красный `#C8102E`
- **Шрифт:** Inter (Google Fonts)
- **Кнопки:** красная `.btn-red` и прозрачная `.btn-ghost`

## Главная страница (index.html)

### Структура сверху вниз

1. **Header** — лого слева, бейдж локации справа (на мобиле скрыт)
2. **Mobile-loc** — бейдж "Производство в г. Актау" только на мобиле (под шапкой)
3. **Hero** — сплит 50/50: текст слева, фото `IMG_0667.PNG` справа
4. **Карточки категорий** — сетка 3×2
5. **Популярная продукция** — 5 карточек товаров (product1–5)
6. **Статистика** — красная полоса (5+ лет, 100+ точек, 5000+ клиентов)

### Hero

- **Desktop/iPad:** grid `50% 50%`, высота `min-height: 520px`
- **Мобил:** `display: block`, текст поверх картинки (z-index 2), картинка абсолютно справа (z-index 1)
- Градиент на картинке: `90deg, rgba(0,0,0,0.95) → transparent` — чтобы текст не залезал на бутылки
- Фото бутылок ярче через `filter: brightness(1.2)` + жёлтый `drop-shadow`

### Карточки категорий (`.cat-grid`)

- Сетка 3 колонки × 2 строки, высота карточки 160px (мобил: 120px)
- Каждая карточка: фоновое фото + тёмный overlay + иконка + название + описание
- На мобиле описание скрыто (`display: none`)
- Карточки: Производство, Опт/Розница, Бытовая химия, Автохимия, Новинки, Каталог

### Мобильная шапка

- Гамбургер (`≡`) вместо кнопки локации
- `.mobile-loc` — тонкий бейдж под шапкой (прозрачный фон, мелкий шрифт)

## Адаптив (breakpoints)

| Брейкпоинт | Поведение |
|---|---|
| `> 1024px` (ПК) | Hero 50/50, 5 карточек продуктов, 3 колонки категорий |
| `768–1024px` (iPad) | Hero 55/45, 3 карточки продуктов в ряд |
| `< 767px` (мобил) | Hero блок (текст + картинка), кнопки 100% ширины, гамбургер |

## Страницы категорий

Структура каждой страницы:
1. Header с логотипом и кнопкой "На главную"
2. Hero-секция с заголовком и кнопкой "Скачать прайс"
3. Карточки продуктов / контент
4. Блок с кнопками скачивания прайс-листа

Кнопка "Скачать прайс (PDF)" ведёт на `#` — заменить на путь к файлу когда будет готов PDF.

## Логотип

Файл: `out/logo.jpg`
Формат: JPG, круглый обрез через CSS `border-radius: 50%; object-fit: cover`

Если обновить логотип — положить новый файл в `out/logo.jpg` и запушить.

## Добавление нового фото в hero или карточки

1. Положить файл в `out/images/`
2. В `index.html` заменить `src` или `background-image` на новое имя файла
3. `git add . && git commit -m "..." && git push`
