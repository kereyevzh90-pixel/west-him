# West Him — сайт-каталог

Сайт казахстанского производителя бытовой и автохимии (г. Актау).
Цель: каталог продукции + информация для оптовых партнёров.

## Деплой

- **Живой сайт:** https://west-him.vercel.app
- **GitHub:** https://github.com/kereyevzh90-pixel/west-him
- Vercel автоматически деплоит при `git push` в `master`
- **Всегда пушить сразу после правок**, не спрашивая пользователя

```bash
git add .
git commit -m "описание"
git push
```

## Дизайн

- **Тема:** светлая (фон `#fff`), акцент красный `#C8102E`
- **Шрифт:** Inter (Google Fonts)
- **Фон страниц:** белый `#fff`, секции разделены границей `1px solid #ebebeb`
- **Кнопки:**
  - `.btn-red` — красная заливка, белый текст ("Скачать прайс")
  - `.btn-outline` — белая с серой обводкой ("Стать партнёром", "Заказать в WhatsApp")
  - `.header-cta` — белая с красной обводкой и красным текстом ("Связаться"), при hover заливается красным
- **Правило:** нет оранжевых, синих, зелёных, фиолетовых акцентов — только красный `#C8102E` и чёрный

## Главная страница (index.html)

### Структура сверху вниз

1. **Header** — лого + "Химическое производство", навигация по центру, кнопка "Связаться" справа
2. **Hero** — фото на весь фон, белый градиент слева, текст поверх
3. **Карточки категорий** — 4 карточки, только фото (текст уже внутри PNG)
4. **FAQ** — аккордеон, 5 вопросов, белый фон
5. **Статистика** — 4 блока, белый фон, красные иконки, тёмные цифры

### Header

- Лого: `logo.jpg` (круглое) + "WEST HIM" + подпись "Химическое производство"
- Навигация: Главная (активная — красная с подчёркиванием), Каталог, Производство, О компании, Партнёрам, Контакты
- Кнопка: телефонная иконка + "Связаться" (outlined красная)
- На мобиле: навигация и кнопка скрыты, показывается гамбургер

### Hero (index.html)

- Фото `IMG_0838.PNG` — фон на **весь** блок hero (`background: cover`)
- Белый градиент слева через `::before`: `linear-gradient(to right, #fff 38%, rgba(255,255,255,0.85) 52%, rgba(255,255,255,0) 72%)`
- Текст `.hero-left` (z-index: 2) — без фонового цвета, читается на белом градиенте
- `.hero-right` — скрыт (`display: none`), фото задаётся только через `.hero`
- Бейдж: `→ Собственное производство в Казахстане` (красный, uppercase)
- Заголовок (3 строки): "Производство" / "**бытовой химии**" (красный, `.red`) / "в Казахстане" — `white-space: nowrap`
- Подпись: "Казахстанский производитель бытовой химии с собственным заводом в Актау. Производим до 30 тонн продукции ежедневно и поставляем по всему Казахстану и СНГ."
- Кнопки: "Каталог продукции →" (`.header-cta` — белая с красной обводкой) + "Стать партнёром" (`.btn-outline`)

### Карточки категорий (`.cat-grid`)

4 карточки — **только PNG-изображения** (иконка, название, описание и стрелка уже нарисованы в самом файле):
- 1-я (Производство) → `IMG_0845.PNG`
- 2-я (Опт/Розница) → `IMG_0847.PNG`
- 3-я (Бытовая химия) → `IMG_0846.PNG`

### FAQ (`.faq-section`)

- Белый фон, аккордеон с 5 вопросами

- Ключевые слова в ответах: `<span class="red">`

- Белый фон, 4 блока в ряд, разделены границами `1px solid #ebebeb`

## Админ-панель

**URL:** https://west-him.vercel.app/admin/
**Файл:** `out/admin/index.html`

### Как работает

- Первый вход: вводишь пароль (придумываешь сам) + GitHub Token → сохраняется в localStorage браузера
- Следующие входы: только пароль
- Кнопка "Сбросить" — сменить пароль или токен
- В коде сайта нет никаких секретов
- После загрузки файла → GitHub получает коммит → Vercel автодеплоит (~1 мин)
- Кнопка "Удалить" удаляет файл из репозитория через GitHub API (тот же токен)

| Раздел | Файл в репо |

| Hero фото — мобиль | `out/images/IMG_0851.PNG` |
| Карточка "Производство" | `out/images/IMG_0845.PNG` |

| PDF Прайс-лист | `out/price.pdf` |

### GitHub Token для панели

Создать на github.com/settings/tokens → Classic → галочка `public_repo` → No expiration.
Токен вводится при первом входе в панель, хранится в localStorage.

## Замена фото вручную (через код)

### Hero на главной
1. Положить файл в `out/images/`
2. В `index.html` изменить `url('images/...')` в CSS-правиле `.hero { background: ... }`
3. `git add . && git commit -m "..." && git push`

### Карточки категорий (главная)
Карточки отображают PNG целиком — текст и иконки нарисованы в самом изображении.
1. Заменить файл в `out/images/` (или добавить новый)
2. В `index.html` изменить `background-image` у нужного `.cat-photo`
3. `git add . && git commit -m "..." && git push`
