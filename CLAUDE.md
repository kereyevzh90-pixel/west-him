# West Him — сайт-каталог

Сайт казахстанского производителя бытовой и автохимии (г. Актау).
Цель: каталог продукции + информация для оптовых партнёров.

## Деплой

- **Живой сайт:** https://west-him.vercel.app
- **GitHub:** https://github.com/kereyevzh90-pixel/west-him
- Vercel автоматически деплоит при `git push` в `master`

```bash
git add .
git commit -m "описание"
git push
```

## Структура проекта

Сайт — статические HTML файлы в папке `out/`. Next.js проект в `src/` не используется для деплоя.

```
out/
├── index.html              ← главная страница (орбитальная анимация)
├── logo.jpg                ← логотип (круглый, молекулярная структура)
├── bytovaya-himiya.html    ← Бытовая химия
├── avtohimiya.html         ← Автохимия
├── proizvodstvo.html       ← Производство
├── opt.html                ← Опт / Розница
└── novinki.html            ← Новинки
```

## Дизайн

- **Тема:** тёмная (фон #000), акцент красный `#C8102E`
- **Шрифт:** Inter (Google Fonts)
- **Градиент заголовков:** `linear-gradient(135deg, #ff4d6b, #ff9a3c)`
- Все 5 страниц категорий используют одинаковую красную цветовую схему

## Главная страница (index.html)

### Секция Hero
- Контент по центру страницы (flexbox, `align-items: center; text-align: center`)
- Бейдж → заголовок → описание → две кнопки

### Секция орбитальной анимации
- Располагается **ниже** героя, отдельная секция `<section class="orb-section">`
- Высота 500px desktop / 440px mobile
- Фон: клеточная сетка (как у левой части)
- 5 узлов вращаются вокруг логотипа в центре
- Клик на узел → переход на страницу категории

### Важно: центрирование орбиты
`.stage` центрируется через JS (`offsetWidth/2`, `offsetHeight/2`), НЕ через CSS `top:50%; left:50%` — потому что stage является нулевым div и CSS-transform не работает:
```js
const cx = orb.offsetWidth / 2;
const cy = orb.offsetHeight / 2;
stage.style.cssText = `left:${cx}px;top:${cy}px;`;
```

## Страницы категорий

Структура каждой страницы:
1. Header с логотипом и кнопкой "На главную"
2. Hero-секция с заголовком и кнопкой "Скачать прайс"
3. Карточки продуктов / контент
4. Блок с кнопками скачивания прайс-листа

Кнопка "Скачать прайс (PDF)" сейчас ведёт на `#` — когда будет готов PDF файл, заменить `href="#"` на путь к файлу.
