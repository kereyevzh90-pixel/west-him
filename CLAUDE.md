# West Him — сайт-каталог

Сайт казахстанского производителя бытовой и автохимии (г. Актау).
Цель: каталог продукции + информация для оптовых партнёров.

## Деплой

- **Живой сайт:** https://west-him.vercel.app
- **GitHub:** https://github.com/kereyevzh90-pixel/west-him
- Vercel автоматически деплоит при `git push` в `master`
- **Всегда пушить сразу после правок**, не спрашивая пользователя

```bash
git add .
git commit -m "описание"
git push
```

## Дизайн

- **Тема:** тёмная (фон `#000`), акцент красный `#C8102E`
- **Шрифт:** Inter (Google Fonts)
- **Кнопки на главной:**
  - `.btn-ghost-red` — прозрачная с красной обводкой и красным текстом ("Связаться с нами")
  - `.btn-ghost` — прозрачная с белой обводкой ("Смотреть каталог")
- **Правило:** нет оранжевых, синих, зелёных, фиолетовых акцентов — только красный `#C8102E` и белый

## Главная страница (index.html)

### Структура сверху вниз

1. **Header** — лого слева (только WEST HIM®, без подписи), бейдж локации справа (на мобиле скрыт)
2. **Mobile-loc** — бейдж "Производство в г. Актау" только на мобиле (под шапкой, по центру)
3. **Hero** — разный макет по брейкпоинтам (см. ниже)
4. **Карточки категорий** — 4 карточки на всех версиях, 2×2 на мобиле
5. **Популярная продукция** — 5 карточек товаров (product1–5)
6. **Статистика** — красная полоса (5+ лет, 100+ точек, 5000+ клиентов)

### Hero

| Версия | Макет | Файл |
|---|---|---|
| ПК (`> 1024px`) | grid `40% 60%`, `min-height: 520px`, cover | `IMG_0667.PNG` |
| iPad (`768–1024px`) | grid `45% 55%`, `min-height: 460px`, contain | `IMG_0740.PNG` |
| Мобил (`< 768px`) | flex-column, текст → фото → кнопки | `IMG_0740.PNG` |

**Мобильный hero** (`< 768px`):
- `display: contents` на `.hero-left` — дети становятся flex-элементами `.hero`
- Порядок через `order`: h1(1) → desc(2) → hero-right(3) → hero-actions(4)
- Текст по центру (`text-align: center`)
- Кнопки в ряд бок о бок (`flex-direction: row`)
- Высота фото: `280px`, `background-size: contain`

### Карточки категорий (`.cat-grid`)

На **всех** брейкпоинтах — 4 карточки (5-я и 6-я скрыты), без overlay, без текста, одни и те же фото:
- 1-я (Производство) → `IMG_0747.PNG`
- 2-я (Опт/Розница) → `IMG_0744.PNG`
- 3-я (Бытовая химия) → `IMG_0745.PNG`
- 4-я (Автохимия) → `IMG_0746.PNG`

Сетка: ПК — 4 колонки, iPad — 2×2, Мобил — 2×2, `aspect-ratio: 16/9`, gap 24px (ПК/iPad) / 12px (мобил).

### Мобильная шапка

- Гамбургер (`≡`) вместо кнопки локации
- `.mobile-loc` — бейдж под шапкой, `width: fit-content`, `margin: auto` (по центру)

## Страницы категорий

### Общая структура каждой страницы

1. Header — лого + кнопка "На главную"
2. **Hero-секция** — фото-фон с оверлеем `rgba(0,0,0,0.82)`, серый бейдж, белый заголовок, кнопка "Скачать прайс"
3. Контент (карточки товаров / условия / особенности производства)
4. Прайс-блок с кнопками

### Правила оформления страниц

- **Фон hero:** фото + `rgba(0,0,0,0.82)` оверлей через `::after`, все дочерние элементы `z-index: 1`
- **Бейдж** (`.cat-badge`): серый — `background: rgba(255,255,255,0.05)`, `border: rgba(255,255,255,0.12)`, `color: rgba(255,255,255,0.85)`
- **Заголовок h1:** полностью белый, `.grad` = `color: #fff`
- **Блобы:** убраны (`display: none`) — фон чёрный
- **Клетки (grid-bg):** убраны
- **Кнопка в hero:** "Скачать прайс" — красная `.btn-red`
- **Мобил:** кнопки CTA на всю ширину, прайс-блок складывается вертикально

## Добавление/замена фото

### Hero на главной (ПК/iPad)
1. Положить файл в `out/images/`
2. В `index.html` изменить `url('images/...')` у `.hero-right` в нужном медиа-запросе
3. `git add . && git commit -m "..." && git push`

### Hero на страницах категорий
1. Положить файл в `out/images/`
2. В нужном `.html` изменить `url('images/...')` у `.hero-cat`
3. Если заменяешь файл с тем же именем — добавь `?v=N` к URL для сброса кеша
4. `git add . && git commit -m "..." && git push`

### Карточки категорий (все версии)
1. Положить файл в `out/images/`
2. В `@media (min-width: 768px)` изменить `url(...)` у нужного `.cat-card:nth-child(N) .cat-bg`
3. Продублировать то же правило в `@media (max-width: 767px)` для мобила
4. `git add . && git commit -m "..." && git push`
