# West Him — сайт-каталог

Сайт казахстанского производителя бытовой и автохимии (г. Актау).
Цель: каталог продукции + информация для оптовых партнёров.

## Деплой

- **Живой сайт:** https://west-him.vercel.app
- **GitHub:** https://github.com/kereyevzh90-pixel/west-him
- Vercel автоматически деплоит при `git push` в `master`

```bash
git add .
git commit -m "описание"
git push
```

## Структура проекта

## Дизайн

- **Тема:** тёмная (фон `#000`), акцент красный `#C8102E`
- **Шрифт:** Inter (Google Fonts)
- **Кнопки:** красная `.btn-red` и прозрачная `.btn-ghost` — `white-space: nowrap`, рядом в одну строку

## Главная страница (index.html)

### Структура сверху вниз

1. **Header** — лого слева, бейдж локации справа (на мобиле скрыт)
2. **Mobile-loc** — бейдж "Производство в г. Актау" только на мобиле (под шапкой)
3. **Hero** — сплит: текст слева, фото справа
4. **Карточки категорий** — 4 карточки (ПК/iPad) или 6 карточек (мобил)
5. **Популярная продукция** — 5 карточек товаров (product1–5)
6. **Статистика** — красная полоса (5+ лет, 100+ точек, 5000+ клиентов)

### Hero

Изображения задаются через CSS `background-image` на `.hero-right`:

| Версия | CSS | Файл |
|---|---|---|
| ПК (`> 1024px`) | базовый CSS | `IMG_0667.PNG` |
| iPad (`768–1024px`) | `@media (max-width:1024px) and (min-width:768px)` | `IMG_0740.PNG` |
| Мобил (`< 767px`) | `@media (max-width:767px)` | `photo_2026-05-11_14-50-30.jpg` |

- **ПК:** grid `40% 60%`, `min-height: 520px`, `background-size: cover`
- **iPad:** grid `45% 55%`, `min-height: 460px`, `background-size: contain`
- **Мобил:** grid `50% 50%`, `height: 380px`, `background-size: contain; background-position: center top`
- Заголовок: `white-space: nowrap` — "АВТО И ДОМА" не переносится
- Кнопки: `flex-wrap: nowrap` — обе в одну строку

### Карточки категорий (`.cat-grid`)

**Мобил** (`< 768px`): 6 карточек, сетка 3×2, высота 120px, с overlay и текстом, оригинальные фото из HTML inline-стилей.

**ПК + iPad** (`≥ 768px`) — переопределяется в `@media (min-width: 768px)`:
- 4 карточки (Новинки и Каталог скрыты через `display: none`)
- Сетка: ПК — 4 колонки, iPad — 2×2
- `aspect-ratio: 16/9`, без overlay, без текста
- Новые фото через CSS `!important` (переопределяют inline-стили):
  - 1-я (Производство → `proizvodstvo.html`) → `IMG_0747.PNG`
  - 2-я (Опт/Розница → `opt.html`) → `IMG_0744.PNG`
  - 3-я (Бытовая химия → `bytovaya-himiya.html`) → `IMG_0745.PNG`
  - 4-я (Автохимия → `avtohimiya.html`) → `IMG_0746.PNG`
- Gap: 24px, padding секции: 52px по бокам

### Мобильная шапка

- Гамбургер (`≡`) вместо кнопки локации
- `.mobile-loc` — тонкий бейдж под шапкой

## Адаптив (breakpoints)

| Брейкпоинт | Hero | Карточки категорий |
|---|---|---|
| `≥ 1025px` (ПК) | 40/60, cover, 5 бутылок | 4 шт, 4 колонки, новые фото |
| `768–1024px` (iPad) | 45/55, contain, iPad-фото | 4 шт, 2×2, новые фото |
| `< 768px` (мобил) | 50/50, 380px, мобил-фото | 6 шт, 3 колонки, оригинальные фото |

## Страницы категорий

Структура каждой страницы:
1. Header с логотипом и кнопкой "На главную"
2. Hero-секция с заголовком и кнопкой "Скачать прайс"
3. Карточки продуктов / контент
4. Блок с кнопками скачивания прайс-листа

Кнопка "Скачать прайс (PDF)" ведёт на `#` — заменить на путь к файлу когда будет готов PDF.

## Добавление/замена фото

### Hero
1. Положить файл в `out/images/`
2. В `index.html` в нужном медиа-запросе изменить `url('images/...')` у `.hero-right`
3. `git add . && git commit -m "..." && git push`

### Карточки категорий (ПК/iPad)
1. Положить файл в `out/images/`
2. В `@media (min-width: 768px)` изменить `url(...)` у нужного `.cat-card:nth-child(N) .cat-bg`
3. `git add . && git commit -m "..." && git push`
